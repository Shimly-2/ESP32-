## ESP32核心板

**打造属于自己的超迷你核心板就是说**

该ESP32核心板尺寸仅有**2.5cm*5.5cm**，且板载**1.14寸全彩LCD**，**64Mb SPI Flash**，采用**TypeC**接口以及**锂电池**双供电模式，可实现无线投屏，实时时钟等多项功能。

### 硬件板载资源

- ESP32（板载天线）
- TypeC接口
- CH340C
- 1.14寸LCD显示屏
- LED
- 轮拨按钮
- 64Mb SPI Flash
- TP4054 锂电池充电管理
- 全I/O引出
